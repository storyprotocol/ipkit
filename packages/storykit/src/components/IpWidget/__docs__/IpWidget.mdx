# IP Asset Widget

IP Asset card widget that displays IP data including the IpGraph, and IpLicenseAccordion.

#### Example

<Canvas of={IpWidgetStory.Select} />

<Controls />

## Usage

```ts
import { IpWidget } from "@storyprotocol/storykit"

const ExamplePage = () => {
  return (
    <IpWidget ipId={"0xbbf08a30b9ff0f717a024a75963d3196aaf0f0dd"} />
  );
};

export default ExamplePage;
```

#### Arguments

- **ipId**: _string_ - The IP Asset Id, an Address starting 0x.
- **isBottomNav**: _boolean_ - Position of Tabs
